/api/v1/send_message Endpoint Documentation

Here is the documentation for the endpoint /api/v1/send_message that you can use to send a message to a requested Discord channel:

Endpoint URL

The endpoint URL is: https://<your_base_url>/api/v1/send_message

HTTP Method

The HTTP method used to send the request to the endpoint is POST.

Request Body

The request body should be in JSON format with the following structure: 

  {
      "message": <string>,
      "discord_webhook": <string>
  }

Request Body Parameters

  • message: A string representing the message content to be sent to the Discord channel.
  • discord_webhook: A string representing the Discord webhook URL to which the message should be sent.

Example Request

Here is an example of a valid request to the endpoint: 

  curl -X POST https://<your_base_url>/api/v1/send_message \
  -H "Content-Type: application/json" \
  -d '{
      "message": "Hello from the API!",
      "discord_webhook": "https://discordapp.com/api/webhooks/<webhook_id>/<webhook_token>"
  }'

Response

If the request is successful, the endpoint will return a JSON response with the following structure: 

  {
      "message": "Message sent successfully!"
  }

Error Responses

In case of any errors, the endpoint will return a JSON response with an error message, for example: 

  {
      "error": "Invalid discord webhook URL"
  }

Notes

  • Ensure that the Discord webhook URL provided is valid.
  • The message content should not exceed 2048 characters.

I hope this documentation helps you in integrating with the endpoint and sending messages to the Discord channel.
